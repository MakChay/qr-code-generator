# QR Code Generator (Python)

A desktop QR Code Generator built with Python and Tkinter.

## Features
- Generate QR codes from text or URLs
- Live QR preview inside the GUI
- Save As functionality
- Simple and beginner-friendly interface

## Requirements
- Python 3.10+
- qrcode
- Pillow

## Run the app
python qr_gui.py
